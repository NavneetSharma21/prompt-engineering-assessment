# Prompt Engineering Assessment Answers

## Exercise 1

Prompt:
"Summarize the following paragraph in one sentence."

Answer:
Zero-Shot Prompting

---

## Exercise 2

Prompt contains one example before asking a new question.

Answer:
One-Shot Prompting

---

## Exercise 3

Scenario:
Generate responses in the same writing style consistently.

Answer:
Few-Shot Prompting

Reason:
Multiple examples help the AI understand and follow the required style.

---

## Exercise 5

Original Prompt:

Make this sound professional:
"Send this now."

Improved Prompt:

Example:

Informal:
Call me later.

Professional:
Please call me at your earliest convenience.

Now rewrite:

"Send this now."

Expected Response:

Please send this at your earliest convenience.

---

## Exercise 6

Statement:

Few-shot prompting is used when the task is very simple.

Answer:

False

Explanation:

Few-shot prompting is useful when examples are needed to improve consistency and accuracy.

---

## Exercise 7

Prompt A

Write a product description.

Prompt B

Write a 3-sentence product description for a budgeting app targeting young professionals.

Answer:

Prompt B

Reason:

It provides context, audience, and output length, resulting in a better AI response.

---

## Exercise 8

Scenario:

Maintain the same writing style across multiple outputs.

Answer:

Few-Shot Prompting

Reason:

Multiple examples teach the AI the desired writing pattern.

---
